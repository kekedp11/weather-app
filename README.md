# weather-app
A simple weather app using HTML, CSS, and JavaScript with OpenWeather API.
## Features
- Search city weather
- Shows temperature
- Display weather condition
## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
## How to use
1. Clone repo
2. Open index.html
3. Enter city name
